# PrestaShop module skeleton

PrestaShop module skeleton used for version >= 1.7.1

## Getting Started

clone repository and add it to modules folder on a PrestaShop system

### Prerequisites

* composer
* PHP >= 5.4

### Installing

1. Execute *composer install*.
2. Install module.
